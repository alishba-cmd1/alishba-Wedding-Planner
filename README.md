# 🎉 Event Planner Website

A modern and responsive event planning website developed using HTML, CSS, and JavaScript. The website provides an elegant interface where users can explore event planning services, browse event packages, and connect with professional event organizers.

## ✨ Features

* Responsive and modern UI
* Event services and packages showcase
* User-friendly navigation
* Interactive and attractive design
* Fast loading and mobile-friendly website

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
